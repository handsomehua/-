go试题

package main 入口文件，可以生成可执行文件

方法定义含义:1.大写包外部可以访问 2.小写只能包内部访问

不建议使用 print println

传值 &表示内存地址 *表示值

局部变量定义不使用会报错，全局变量可以定义不使用

什么时候使用常量，当声明一个值不会被修改，才适合使用常量

fmt.Scan和fmt.Scanln区别就是一个没有输完回车会一直等待，另一个回车就算全部输完了

fmt.Scan和fmt.Scanln和fmt.Scanf有一个缺陷，无法获取空格后的内容

fmt.Sprintf和fmt.Printf的区别：一个返回拼接字符串不打印，另一个返回拼接字符串并打印

int8只能和int8相加或比较，int16 int32同理，不相同必须强制转换后在相加或比较

指针是为了节省内存

什么是指针？指针，是一种数据类型，用于表示数据的内存地址

浮点型加减乘除会丢失精度，解决办法使用decimal包

字节集合[232 146 139 233 131 180 229 141 142]字节集合可以转中文，中文也可以转字节集合

rune的集合[33931 37108 21326]rune的集合可以转中文，中文也可以转rune的集合

字符串本质上不可以被修改，修改不会修改原来的值，而是在生成一个值，老的值不会变

make只用于 切片,字典类型map,channel

推荐使用make创建切片：因为指定了容量就默认有了这么多存储位置，就不用在扩容

切片容量指存储的个数

切片扩容后容量则会翻一倍

切片不是数组

通过切片切出来的数据和原切片地址是同一个地址

切片通过append追加元素如果扩容了，和原切片不再是同一个地址

切片和数组的区别？:切片在定义的时候可以定义长度和容量，在添加数据的时候可以自动扩容，数组在声明就定死了

new和make的区别？:new返回的是一个指针，make返回的是一个对象

map只声明是无法给里面添加row["name"] = 888值的，只能用于整体赋值

map键不能重复并且键必须可哈希(int/bool/float/string/array)

map无论是否存在扩容都指向同一个地址

什么是结构体？结构体是一个复合类型，用于表示一组数据

结构体可以定义在函数内也可以定义在函数外

结构体定义首字母大写

结构体指针存储流程：结构体指针开辟两块内存，一块存值的地址，一块存值，而初始化结构体是开辟一块内存存储值

结构体赋值p2 := p1是内部将p1重新拷贝一份给p2

结构体指针赋值p1 := &Person{"武",18} p2 := p1是将指针复制一份给p2,所以当p1重新赋值,p2也会变

...变长参数只能放在最后且只能有一个

多个defer挺迟执行,顺序是倒序执行

强制结构体工厂，分为公有和私有

GO语言数据类型有：整型 浮点型 布尔型 字符串 数组 指针 切片 字典 结构体 接口 特殊

json序列化结构体字段首字母必须大写，不大写必须加标签 `json:"name"`

协程可以实现并行，也就是让多个任务并行执行

使用goroutine协程运行速度快三倍左右

channel管道赋值是引用

channel管道被取值后可以在赋值

通过for循环遍历管道可以不关闭，通过for range遍历管道必须关闭

channel是一种类型，一种引用类型

使用select多路复用不需要关闭channel

读写锁，读同时可以读多个，写只能执行完一个才能下一个

空接口接收参数是空接口类型，运算需要使用类型断言
